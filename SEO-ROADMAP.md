## Publishing Cadence (Daily SEO Growth)

### Every Day
- Publish 1 short insight post (600-900 words) targeting one long-tail keyword.
- Update 1 existing page with new proof: metrics, screenshots, or deployment notes.
- Add internal links from the newest post to 2 service pages and 1 case study.

### Every Week
- Publish 1 detailed case study (1200+ words).
- Refresh sitemap `lastmod` values.
- Track search queries in Google Search Console and create pages for rising terms.

### Every Month
- Consolidate overlapping posts into stronger pillar pages.
- Improve CTR by rewriting meta titles/descriptions for pages with high impressions but low clicks.
- Add FAQ sections for pages getting repeated search intent.

## On-Page Rules
- 1 primary keyword per page in title, H1, first paragraph, URL slug.
- Use semantic headings (`h2`, `h3`) with secondary keywords naturally.
- Add 3-5 internal links per page.
- Add one clear CTA above the fold and one near the end.

## Technical SEO Checklist
- Keep `robots.txt` and `sitemap.xml` updated.
- Keep canonical tags on indexable pages.
- Add Open Graph + Twitter metadata to each page.
- Use JSON-LD (`Organization`, `WebSite`, `Article`, `FAQPage`, `BreadcrumbList`) where relevant.
- Return true `404` for unknown URLs (avoid soft 404).
